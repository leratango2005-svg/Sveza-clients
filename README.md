# Sveza — История цен клиентов

Готовый проект на HTML, CSS, JavaScript и Node.js + Express.

## Что умеет сайт

- показывает историю цен по клиентам;
- фильтрует по клиенту, сорту, размеру и диапазону дат;
- позволяет добавлять новые записи;
- сохраняет данные в JSON-файл;
- нормализует имена клиентов, сорта и размеры;
- устойчив к ошибкам ввода и показывает сообщения на русском языке.

## Структура проекта

- `public/index.html` — интерфейс
- `public/style.css` — стили
- `public/script.js` — логика фронтенда
- `server.js` — backend на Node.js + Express
- `data/db.json` — хранилище данных
- `package.json` — зависимости проекта

## Запуск

1. Откройте папку проекта в терминале.
2. Установите зависимости:

```bash
npm install
```

3. Запустите сервер:

```bash
npm start
```

4. Откройте в браузере:

```text
http://localhost:3000
```

## Важные детали

### Нормализация имен клиентов

- убираются лишние пробелы;
- регистр не учитывается;
- похожие русские и английские буквы приводятся к единому виду.

### Нормализация сортов

Поддерживаются сорта:

- `B/B`
- `B/BB`
- `S/BB`
- `B/CP`
- `B/C`
- `BB/BB`
- `BB/CP`
- `BB/C`
- `CP/CP`
- `CP/C`
- `C/C`

### Нормализация размеров

- `x`, `х`, `*`, `×` приводятся к одному формату;
- пробелы игнорируются;
- размеры вида `2500x1250` и `1250x2500` считаются одинаковыми;
- для типовых размеров есть общие канонические значения.

## Где хранятся данные

Все записи сохраняются в файл:

```text
data/db.json
```

При желании этот проект можно легко перевести с JSON на SQLite.
